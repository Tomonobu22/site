About Me

Hi, I'm Tomonobu.
Software Developer.

I began my journey in Mechatronics Engineering, where my curiosity for how things work naturally evolved into a passion for how software brings those things to life. During my studies, courses in Software for Engineering and Microcontrollers introduced me to programming in C and assembly—an experience that sparked a lasting interest in building and problem-solving through code.

After graduating, I chose to dive deeper into software development, continuing to learn independently before moving to Canada to formalize my skills with a Diploma in Computer Programming in 2022.

Today, I work as a Software Developer with over 3 years of experience specializing in .NET web development. I enjoy building reliable, scalable, and maintainable web applications using technologies such as ASP.NET Core MVC, Web APIs, Razor Pages, and SQL databases. I'm passionate about writing clean, efficient code and continuously improving my skills by exploring modern development practices and technologies.

I also have hands-on experience with automated testing for both web and mobile applications, ensuring the software I build is not only functional, but robust and maintainable.
